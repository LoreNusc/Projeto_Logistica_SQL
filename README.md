# Projeto_Logistica_SQL
📦 Case: SQL LogiTrack Identifição de falha crítica em banco de dados, recuperação dos registros via logs de e-mail (SMTP). Reconstrução da estrutura no SQL Server usando DDL (modelagem técnica), DML (restauração de dados) e DQL (filtros de risco financeiro). Impacto: restabelecimento operacional e proteção de cargas de alto valor.
